java 10 新特性

一直使用 JAVA8，用了好久好久了，最近突然有想法，能不能把后边所有版本的新特性总结一下呢？
说干就干，今天第二篇 JAVA10 新特性。

一、集合新增 copyOf 方法

二、var 关键字(局部变量类型推断)

新增 var 关键字，只能用于局部变量，不能用于方法的参数、方法返回值、类的字段等。

三、总结

以上就是开发者可以使用的JAVA 10 新特性了， 因为是个过渡版本，所以实际新增的特性并不多，希望能够帮助到大家，如果有什么问题，直接联系我们。
